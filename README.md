## yue

|             | none | guttural | dental | dental | labial |
| ----------: | :--- | :------- | :----- | :----- | :----- |
|      tenuis | q ∅  | k g      | t d    | c r    | p b    |
|   aspirated |      | kx gx    | tx dx  | cx rx  | px bx  |
|   fricative |      | x h      | s z    |        | f w    |
|       nasal |      | qŋ ŋ     | qn n   |        | qm m   |
| approximant |      |          | ql l   |        |        |
